AHC TreeAligner
===========


Install
-------

You need following tools to build TreeAligner.
* `GCC 4.7` or later
* `Boost 1.49` or later
* `autotools`

You simply run below:

    cd /path/to/treealigner
    autoreconf -i
    ./configure
    make
    (sudo) make install
